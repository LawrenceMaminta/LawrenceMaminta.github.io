City Pollution Reduction Plan

This application helps you identify the most cost-effective combination of pollution mitigation projects for the city of Greenvale. The concept of Linear Programming and Simplex method was applied in creating this application. This application was developed using R programming language, particularly RShiny.

STRINGS: A New York Times Game Replica

This is a puzzle game (done in terminal) that consists of a group of words in an 8x6 grid. The words follow one theme with a spangram that traverses from one side of the board to the other side. The game ends when all the theme words in the puzzle have been found (win case) or when the player decides to reveal all the words (lose case). This game was developed using Python.
